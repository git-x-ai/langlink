## 🚀 What is LangLink?

LangLink is a modern desktop application built with Electron that provides an intuitive interface for discovering, selecting, and downloading enhanced `.cursorrules` files for Cursor AI. Instead of manually browsing through hundreds of rule files, this app lets you easily find and download the perfect development rules for your project.

**Stop searching, start coding!** With 174+ curated rules across every major framework and technology, LangLink makes it effortless to supercharge your Cursor AI experience.

## ✨ Features

### 🎯 Smart Rule Discovery
- **174+ Enhanced Rules**: Comprehensive collection organized by category
- **Instant Search**: Find rules by technology or framework in seconds
- **Visual Browsing**: Beautiful rule cards with descriptions and file counts
- **Category Organization**: Rules grouped by frontend, backend, mobile, testing, and more

### 📁 Easy File Management
- **One-Click Download**: Select and download multiple rules at once
- **Auto Organization**: Files saved to `Desktop/.cursor/rules/` automatically
- **Drag & Drop Ready**: Simply drag the folder into your project root
- **Visual Indicators**: See which rules you've already downloaded
- **Bulk Operations**: Download entire categories or custom selections

### 🎨 Modern Interface
- **Dark Theme**: Elegant dark mode with smooth animations
- **Responsive Design**: Works beautifully on any screen size
- **Custom Window Controls**: Native-feeling UI with gradient styling
- **Smooth Scrolling**: Lenis integration for buttery-smooth navigation
- **Professional Gradients**: Eye-catching design that's easy on the eyes

### 🔧 Developer-Friendly
- **Zero Configuration**: Works out of the box, no setup required
- **Cross-Platform**: Windows, macOS, and Linux support
- **Offline Ready**: Browse and manage downloaded rules offline
- **TypeScript Powered**: Type-safe codebase for reliability
- **Modern Tech Stack**: React 18, Electron, Tailwind CSS

## 🎮 How to Use

### Quick Start Guide

1. **📥 Download and Install**
   - Download the appropriate installer for your system
   - Run the installer and follow the setup wizard
   - Launch LangLink from your desktop or start menu

2. **🔍 Browse and Search**
   - Use the search bar to find specific technologies (e.g., "React", "Python")
   - Browse categories: Frontend, Backend, Mobile, Testing, etc.
   - Click on rule cards to see detailed descriptions

3. **✅ Select Rules**
   - Check the boxes next to rules you want to download
   - Select multiple rules for bulk download
   - See the download button appear at the bottom

4. **⬇️ Download and Use**
   - Click "Download Selected Rules"
   - Files are saved to `Desktop/.cursor/rules/`
   - Drag the entire `.cursor` folder into your project root
   - Start coding with enhanced Cursor AI assistance!

### 💡 Pro Tips

- **🎯 Stack-Specific**: Download multiple rules for complex stacks (e.g., Next.js + TypeScript + Tailwind)
- **👥 Team Sharing**: Share the `.cursor` folder with your team for consistent AI assistance
- **📝 Version Control**: Keep your rules in git for team-wide consistency
- **🔄 Stay Updated**: Re-download rules periodically as they get enhanced
- **🎨 Custom Combos**: Mix and match rules from different categories
